Other Bots Help Pages

These bots are not so significant as Flapplin, but they're worth mentioning.

Antimatter

  • a.help - Shows the list of Commands.
  • a.add - Add Antimatter.
  • a.prestige - Perform a Prestige, considering you have enough Antimatter.
  • a.galaxy - Sacrifice all Prestiges to Create a Galaxy, if you have enough Antimatter.
  • a.stats - View the Current State of The Universe.
  • a.plans - List of Planned Features.
  • a.stc
  • a.time

EconBot

  • e!help [page] - Shows this message
  • e!work - Work to get money
  • e!bal - Check your balance
  • e!leaderboard/lb - User leaderboard
  • e!transfer [home] [amount] - Transfer money to other user
  • e!changelog - View changelog of the current version
  • e!emojiguess/emo - Emoji-guessing game
  • e!wordguess/wg - Word-guessing game
  • e!room/r - Move the bot to the bot room and go back
  • e!shop - Show the list of items
  • e!buy [item] - Buy an item
  • e!sell [item] - Sell an item
  • e!inventory/inv - Show the list of items you have
Events [Only active through December 1 - January 3]
  • e!event - Current event info
  • e!search - Find the Christmas decorations
  • e!deco - How much Christmas decorations you have
  • e!lb-event - Event leaderboard

Crimson Witch / Electric Archmage

All commands use either ✨ or ⚡, depending on the form.

  • ✨help/commands - Crimson Witch doesn't believe that she's a bot. Whatever, sends the list of comamnds.
  • ✨whoami - I'm Crimson Witch... or Electric Archmage.
  • ✨time/date - Sends the local time.

🐬 Dauphin

  • d.help - Shows this message.
  • d.ask [something] - Generate a response. Not very smart, and the generation time is slow.
  • d.about - About this chatbot.
