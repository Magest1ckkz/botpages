EconBot

  • e!help [page] - Shows this message
  • e!work - Work to get money
  • e!bal - Check your balance
  • e!leaderboard/lb - User leaderboard
  • e!transfer [home] [amount] - Transfer money to other user
  • e!changelog - View changelog of the current version
  • e!emojiguess/emo - Emoji-guessing game
  • e!wordguess/wg - Word-guessing game
  • e!room/r - Move the bot to the bot room and go back
  • e!shop - Show the list of items
  • e!buy [item] - Buy an item
  • e!sell [item] - Sell an item
  • e!inventory/inv - Show the list of items you have

Crimson Witch / Electric Archmage
